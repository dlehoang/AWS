### Week 4 Objectives:

* Understand how to build highly available, fault-tolerant, and scalable architectures on AWS.
* Learn how to monitor infrastructure performance and track application resource utilization.

### Week 4 Achievements:

Grasped core cloud design principles for disaster recovery, fault tolerance, and multi-region domain routing using Amazon Route 53.

Understood how Elastic Load Balancing effectively distributes incoming application traffic across multiple target EC2 instances.

Successfully configured AWS Auto Scaling to dynamically provision or terminate EC2 instances based on real-time traffic demands.

Mastered system monitoring with Amazon CloudWatch by creating custom dashboards and configuring email alert notification rules via CloudWatch Alarms.
