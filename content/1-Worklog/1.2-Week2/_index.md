### Week 2 Objectives:

* Understand and practice deploying virtual servers and setting up cloud development environments.
* Master cloud data storage concepts and hosting static websites on AWS.

### Tasks to be carried out this week:
| Day | Task | Start Date | Completion Date | Reference Material |
| --- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------- | --------------- | ----------------------------------------- |
| 2   | - Learn about AWS IAM (Identity and Access Management) to manage users, groups, and permissions. <br> - Practice setting up Multi-Factor Authentication (MFA) for Root and IAM accounts. | 27/04/2026  | 27/04/2026      | <https://cloudjourney.awsstudygroup.com/> |
| 3   | - Study Amazon EC2 fundamentals: Instance types, Amazon Machine Images (AMI), and Elastic Block Store (EBS) volumes. <br> - Practice launching, configuring, and connecting to an EC2 instance via SSH. | 28/04/2026 | 28/04/2026      | <https://cloudjourney.awsstudygroup.com/> |

### Week 2 Achievements:

Mastered the Principle of Least Privilege by successfully creating and managing secure IAM Users, Groups, and custom Policies.

Successfully deployed and managed virtual servers (EC2 instances) and understood how to attach block storage volumes (EBS) for resource scaling.

Familiarized with the cloud-based development environment using AWS Cloud9 IDE for running and deploying cloud applications.

Successfully initialized Amazon S3 buckets, configured web-accessible policies, and hosted a functional static website securely on the public internet.
